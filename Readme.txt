# Finance Tracker App

## Overview

The Finance Tracker App is a simple financial management tool with a graphical user interface. It allows users to record income and expenses, view their financial entries, calculate totals, and perform other related actions. The application is built using Python and the Tkinter library for the graphical interface.


## Dependencies

This project requires Python 3.


## Features

- Record new financial entries (income or expense) with type, amount, and category.
- View a list of all recorded entries.
- Calculate total income, total expenses, and net income.
- View a summary of entries for a specific month.
- Save data to a file and load data from a file.

## How to Use

1. Recording Entries:
   - Enter the type (income or expense), amount, and category in the provided fields.
   - Click the "Record a new entry" button to save the entry.

2. Viewing Entries:
   - Click the "View all entries" button to see a list of all recorded entries.

3. Calculating Totals:
   - Click the "Calculate totals" button to get the total income, total expenses, and net income.

4. Viewing Monthly Summary:
   - Enter the month (1-12) and click the "View summary for a specific month" button.

5. Saving and Loading Data:
   - To save data, enter a filename and click the "Save data to file" button.
   - To load data, enter an existing filename and click the "Load data from file" button.

6. Exiting the App:
   - Click the "Exit" button to close the application.
